[![Latest Version on Packagist](https://img.shields.io/packagist/v/devdojo/database.svg?style=flat-square)](https://packagist.org/packages/devdojo/database)

![GitHub Actions](https://github.com/devdojo/database/actions/workflows/main.yml/badge.svg)

A Livewire-powered database browser for Laravel. Lists your application's tables and lets you search, sort, paginate, and edit records inline.

## Installation

You can install the package via composer:

```bash
composer require devdojo/database
```

## Usage

Drop the Livewire components onto any page:

```blade
@livewire('devdojo.tables')
@livewire('devdojo.database')
```
```

### Testing

```bash
composer test
```
